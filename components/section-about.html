ABOUT PROJECT

Onk Hub

Onkbaonk Hub adalah platform blogging minimalis berbasis static architecture yang dirancang untuk menghadirkan pengalaman publikasi modern tanpa backend tradisional. Project ini menggunakan GitHub API sebagai storage engine dengan sistem token-based publishing untuk memastikan hanya owner yang dapat membuat, mengedit, dan menghapus konten.

⚡ Fast Rendering

Menggunakan sharded JSON architecture untuk memuat blog index secara ringan dan efisien.

🔐 Secure Access

Tidak menggunakan login tradisional. Seluruh akses edit diamankan menggunakan GitHub personal token.

☁️ Cloud Native Storage

Semua data tersimpan dan terversioning otomatis melalui Git-based storage.
